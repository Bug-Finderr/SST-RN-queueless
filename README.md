## Quick Start

```bash
# Backend
cd backend && bun install && cp .env.example .env
# Set MONGO_URI, DB_NAME, JWT_SECRET
bun run dev

# Frontend
cd frontend && bun install && cp .env.example .env
# Set EXPO_PUBLIC_BASE_URL
bun run start
```

## Features

**User:** Login, browse services, book token, view position/wait time, cancel

**Admin:** Create services, call next, skip, complete tokens

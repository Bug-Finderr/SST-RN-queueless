# QueueLess API

Queue/token management system — Bun + MongoDB.

## Quick Start

```bash
bun install
bun run dev          # Bun.serve (default)
bun run dev:express  # Express alternative, install express to use
```

## Env Vars

| Variable | Required | Default |
|----------|----------|---------|
| `MONGO_URI` | Yes | - |
| `DB_NAME` | Yes | - |
| `JWT_SECRET` | Yes | - |
| `JWT_EXPIRES_IN` | No | 24h |
| `PORT` | No | 8000 |

```bash
# Generate JWT secret
openssl rand -base64 32
# or
bun -e "console.log(require('crypto').randomBytes(32).toString('base64'))"
```

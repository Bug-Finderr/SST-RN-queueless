# QueueLess Mobile

Expo app — queue/token booking system.

## Quick Start

```bash
bun install
cp .env.example .env  # Set EXPO_PUBLIC_BASE_URL
bun start
```

## Env Vars

| Variable | Required | Example |
|----------|----------|---------|
| `EXPO_PUBLIC_BASE_URL` | Yes | `http://192.168.1.x:8000` |
